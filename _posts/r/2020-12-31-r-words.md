### R has 9 reserved letters

These cannot be used for variable names

------|---------
c( )  |	Combine
t( )  |	Transpose (matrices)
q( )  | Quit
/t    | Tab
C( )  | Sets a contrast attribute
D( )  | Computes derivatives
I( )  | Inhibits a change in the class of a value
F     | FALSE
T     | TRUE
